# **CalcuMath**

**CalcuMath** es una herramienta educativa desarrollada en **C#** como proyecto de la asignatura de **Cálculo I**. Su objetivo es apoyar el aprendizaje de conceptos matemáticos mediante una aplicación interactiva.

---

## **Requisitos del Sistema**

Antes de ejecutar el proyecto, asegúrate de tener instalado:

* **Visual Studio 2022** o una versión compatible.
* ** EN ASP .NET ** 
* Los componentes necesarios para ejecutar aplicaciones desarrolladas en **C#**.

> **Nota:** No es necesario instalar **XAMPP, MySQL ni SQL Server**, ya que la aplicación no utiliza una base de datos.

---

## **Abrir el Proyecto**

1. Abre **Visual Studio**.
2. Selecciona **Abrir un proyecto o solución**.
3. Busca la carpeta del proyecto **proyecto**.
4. Abre el archivo de solución con extensión:

```text
ProyectoCalculadora.sln
```

---

## **Restaurar las Dependencias**

Visual Studio restaurará automáticamente las dependencias necesarias del proyecto.

verifica que estén instalados correctamente desde el **Administrador de paquetes NuGet**.

## **Dependencias**

El proyecto utiliza paquetes NuGet para realizar operaciones matemáticas avanzadas.


La aplicación se ejecutará directamente en el entorno de **Visual Studio**.

## **Uso de la Herramienta**

**proyectoCalculadora** es una herramienta educativa desarrollada en **C#** para apoyar el aprendizaje de los contenidos de la asignatura de **Cálculo I**.

La aplicación puede incluir diferentes herramientas educativas como:

* **Calculadora de límites**.

## **Funcionamiento del Sistema**

La aplicación funciona de manera **local** y no requiere conexión a Internet ni una base de datos externa.

Los cálculos y operaciones matemáticas se realizan directamente mediante la lógica programada en **C#**.

La información necesaria para el funcionamiento de la aplicación se encuentra integrada dentro del propio proyecto.

## **Solución de Problemas**

### **El proyecto no inicia**

* Verifica que **Visual Studio** esté instalado correctamente.
* Comprueba que tengas instalada la versión correcta de **ASP .NET**.
* Revisa los errores mostrados en la ventana **Error List** de Visual Studio.

### **Error relacionado con paquetes o librerías**

* Abre el **Administrador de paquetes NuGet**.
* Restaura los paquetes del proyecto.
* Verifica que todas las referencias estén instaladas.

### **Los cálculos no funcionan correctamente**

* Verifica los datos ingresados por el usuario.
* Comprueba la lógica matemática implementada.
* Revisa las funciones y métodos encargados de realizar los cálculos.
